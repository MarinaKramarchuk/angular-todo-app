# 🚀 Modern Angular To-Do App

A professional-grade technical assessment project demonstrating mastery of the **Angular (v20+)** ecosystem. This application showcases high-performance reactivity, clean architecture, and modern styling patterns.

## 🌟 Live Demo

🔗 [View Project on GitHub Pages](https://MarinaKramarchuk.github.io/angular-todo-app/)

---

## 🛠 Tech Stack

- **Framework:** Angular 20+ (Standalone Components)
- **State Management:** Angular **Signals** (Writables, Computeds, Effects)
- **Change Detection:** **Zoneless** (ExperimentalZonelessChangeDetection)
- **Styling:** **SCSS** (Modular & Responsive)
- **Tooling:** Angular CLI, GitHub Pages CI/CD

## ✨ Key Features (Advanced Patterns)

- **Signal-based Reactivity:** Optimized data flow using `signal()`, `computed()`, and `effect()`.
- **Modern Control Flow:** Implementation of `@if`, `@for`, and `@empty` block syntax.
- **Component Architecture:** Strict separation between Smart (logic) and Dumb (UI) components.
- **Persistence:** Seamless synchronization with Browser LocalStorage.

## ⚙️ Local Development

1. Clone: `git clone https://github.com/MarinaKramarchuk/angular-todo-app.git`
2. Install: `npm install`
3. Launch: `npm start`
