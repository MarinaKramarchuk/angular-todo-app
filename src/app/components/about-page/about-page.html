About the Project

Welcome! This is TaskFlow, a modern Task Management application built to demonstrate the power of Angular 21.

The main goal of this pet project was to implement a clean, reactive, and highly performant user interface using the latest industry standards.

Core Features:

  • Reactive State: Managed entirely via Angular Signals for optimal performance.
  • Routing: Seamless Single Page Application (SPA) experience.
  • Global Error Handling: Centralized messaging system via dedicated services.
  • Responsive Design: Fully mobile-friendly UI powered by Bulma CSS.

Technologies Used:

Angular 21 TypeScript Signals Bulma CSS RxJS
